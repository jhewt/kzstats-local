KZStats Local

KZStats for server owners

Download this project as a .zip file Download this project as a tar.gz file

Installation

  1. Download latest release here.
  2. Extract the .zip file and move the contents to a directory on your webserver, naming it for example "stats".
  3. Open up http://www.yoursite.com/stats/install/ on your browser and fill out the form.
  4. Your stats page should now be working. If something isn't right, read the section below.

Troubleshooting

If the installer doesn't work, you can manually edit your correct values in the api/settings.example.php file and rename it to settings.php.
