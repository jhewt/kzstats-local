KZStats Local

KZStats for server owners

Download this project as a .zip file Download this project as a tar.gz file

IMPORTANT

Please use this link to download the latest release. Other links shown on the right (the .zip and .tar.gz icons) are for the sources only.

Requirements

  • PHP >5.4
  • MySQL
  • Apache with mod_rewrite (other HTTP-servers may work as well)

Installation

  1. Download latest release here.
  2. Extract the .zip file and move the contents to a directory on your webserver, naming it for example "stats".
  3. Open up e.g. http://www.yoursite.com/stats/install/ on your browser and fill out the form.
  4. Your stats page should now be working. If something isn't right, read the section below.

Troubleshooting

  • If the installer doesn't work, be sure to change permissions of the /api/ directory to 644. If all else fails, you can also manually edit the correct values in the api/settings.example.php file and rename it to settings.php.

  • If you use other any HTTP-server, replace the .htaccess file on the api directory with your webserver's own configuration to redirect all requests to index.php.
